# Projects
## Deep Learning based American Sign Language Recognition
Developed three classes of Artificial Neural Network (ANN) models using the PyTorch framework to translate American Sign Language (Kaggle dataset) into a text-based representation. The first class is a single generic Multilayer Perceptron (MLP) network. In contrast, the second class is two untrained CNNs (Convolutional Neural Networks)- a baseline CNN and a custom CNN (with Adam optimizer and L2 regularization). Finally, the third class consists of two pre-trained CNNs (GoogLeNet and ResNet18 networks), mainly used to leverage transfer learning. All five models were tested using images captured in real-time and achieved an accuracy of above 90%.
